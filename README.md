# 🌤️ Weather App

A simple, responsive weather application built with Vanilla JavaScript and the OpenWeatherMap API. This project allows users to search for real-time weather data by city name.

##  Features
* **Real-time Search:** Get current temperature, humidity, and wind speed for any city.
* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **Live API Integration:** Uses the OpenWeatherMap API for accurate, up-to-date data.

##  Built With
* **HTML5** - Structure and layout.
* **CSS3** - Custom styling and Google Fonts.
* **JavaScript (ES6)** - Fetch API for asynchronous data handling.
* **API** - [OpenWeatherMap](https://openweathermap.org/)
